이용약관

Convert4U 서비스 이용 조건

마지막 업데이트: 2024년 11월 6일
시행일: 2024년 11월 6일

제1조 목적

이 약관은 Convert4U(이하 "회사") 및 회사의 자회사가 제공하는 온라인 파일 변환 서비스(이하 "서비스")의 이용 조건 및 절차를 규정함을 목적으로 합니다.

제2조 정의

2.1 용어의 정의

  • 서비스: Convert4U가 제공하는 모든 파일 변환 기능
  • 사용자: 서비스에 접속하여 이용하는 개인 또는 단체
  • 파일: 사용자가 업로드하는 PDF, 이미지, 영상, 음성, 문서 등의 디지털 콘텐츠
  • 변환: 파일의 형식을 다른 형식으로 전환하는 행위
  • 계정: 관리자용 로그인 계정

제3조 서비스의 범위

3.1 제공되는 서비스

  • PDF 문서 변환 (Word, Excel, PowerPoint, 이미지)
  • 이미지 형식 변환 (JPG, PNG, WEBP, HEIC)
  • 오디오 형식 변환 (MP3, WAV, OGG, M4A, AAC)
  • 비디오 형식 변환 (MP4, MOV, WebM, MKV)
  • PDF 관리 기능 (병합, 분할, 압축)

3.2 서비스의 중단

회사는 다음의 경우 사전 공지 또는 사후 통지로 서비스를 중단할 수 있습니다:

  • 서비스 유지보수
  • 시스템 장애 또는 긴급 상황
  • 정부의 요청 또는 법적 의무

제4조 이용 조건

4.1 접근 방법

  • 회사 웹사이트 접근 (로그인 불필요)
  • 파일 선택 후 변환 형식 선택
  • 변환 완료 후 파일 다운로드

4.2 기술 요구사항

  • 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge)
  • 안정적인 인터넷 연결
  • JavaScript 활성화

4.3 파일 크기 제한

  • 단일 파일 최대 크기: 50MB
  • PDF 병합 최대 크기: 100MB
  • 초과 시 오류 메시지 표시

제5조 사용자의 책임

5.1 금지 행위

사용자는 다음의 행위를 할 수 없습니다:

  • 저작권 보호 파일을 무단으로 변환 또는 배포
  • 서버에 대한 공격 또는 부하 테스트
  • 자동화된 스크래핑 또는 봇 사용
  • 악성 코드 또는 바이러스 포함 파일 업로드
  • 불법 또는 음란 콘텐츠의 변환
  • 다른 사용자의 권리 침해

5.2 개인 책임

사용자는 업로드하는 파일의 합법성과 적절성에 대해 모든 책임을 집니다.

제6조 파일 보안 및 삭제

6.1 파일 처리

  • 변환 완료 후 원본 파일은 즉시 삭제
  • 변환된 파일은 10분 후 자동 삭제
  • 중간 임시 파일은 작업 완료 후 정리

6.2 데이터 안전성

회사는 업계 표준의 보안 조치를 취하지만, 완벽한 보안을 보장할 수 없습니다. 사용자는 민감한 정보를 신중하게 처리해야 합니다.

6.3 백업

변환된 파일을 보관하려면 다운로드 후 보관해야 합니다. 회사는 삭제된 파일에 대한 복구 책임을 지지 않습니다.

제7조 지적재산권

7.1 회사의 권리

Convert4U 서비스 및 관련 모든 콘텐츠, 소프트웨어는 회사의 저작권으로 보호됩니다.

7.2 사용자의 권리

사용자는 변환된 파일에 대해 완전한 소유권을 유지합니다. 단, 저작권이 있는 콘텐츠의 경우 저작권자의 권리는 영향을 받지 않습니다.

7.3 라이선스

회사는 서비스 이용을 위해 제한적인 라이선스를 부여합니다. 무단 복제, 수정, 판매는 금지됩니다.

제8조 면책 조항

8.1 서비스 "있는 그대로" 제공

회사는 다음에 대해 책임을 지지 않습니다:

  • 변환 품질의 완벽성
  • 서비스 중단으로 인한 손실
  • 사용자 파일의 손실 또는 손상
  • 데이터 유출로 인한 간접 손해
  • 제3자 서비스(FFmpeg, LibreOffice 등)의 오류

8.2 손해배상 제한

회사가 부담할 총 책임은 사용자가 회사에 지불한 금액 또는 100달러 중 작은 금액으로 제한됩니다.

제9조 제3자 링크 및 서비스

9.1 외부 링크

Convert4U는 외부 웹사이트에 대한 링크를 제공할 수 있습니다. 회사는 외부 사이트의 콘텐츠, 정확성, 합법성에 대해 책임을 지지 않습니다.

9.2 제3자 서비스

서비스는 다음의 제3자 서비스를 사용합니다:

  • Cloudflare R2: 파일 저장소
  • LibreOffice: 문서 변환
  • FFmpeg: 미디어 처리

이들 서비스의 이용약관은 각 서비스 제공자를 통해 확인할 수 있습니다.

제10조 약관의 변경

10.1 변경 권리

회사는 언제든지 이 약관을 변경할 수 있습니다.

10.2 공지 방법

  • 웹사이트에 30일 전 공시
  • 중대한 변경사항은 이메일 공지
  • 변경 후 계속 서비스를 이용하면 동의한 것으로 간주

제11조 서비스 중단 및 삭제

11.1 계정 삭제

사용자는 언제든지 서비스 이용을 중단할 수 있습니다.

11.2 위반 시 계정 폐기

회사는 약관 위반 시 사전 통지 없이 계정을 폐기할 수 있습니다.

제12조 준거법 및 분쟁 해결

12.1 준거법

이 약관은 사용자 거주 국가의 법률에 따릅니다.

12.2 분쟁 해결

  • 모든 분쟁은 먼저 협의를 통해 해결
  • 협의 실패 시 중재 또는 소송
  • 법적 대리인 비용은 책임 있는 당사자가 부담

제13조 기타

13.1 완전한 합의

이 약관은 Convert4U 서비스 이용에 관한 완전한 합의를 구성합니다.

13.2 분리가능성

약관의 일부가 무효인 경우, 나머지 조항은 계속 유효합니다.

13.3 권리 포기

회사가 약관 위반을 즉시 제재하지 않아도 권리를 포기한 것이 아닙니다.
